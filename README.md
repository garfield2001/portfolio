# Basiren Portfolio

A modern, professional single-page portfolio for a BSIT graduate, web developer, part-time college instructor, IT/CS educator, and creative performer. This first version uses plain HTML, CSS, and JavaScript so it can run from disk, deploy anywhere, and stay easy to edit.

## Why static first

Use HTML/CSS/JS for this portfolio while it is mostly presentation, project proof, contact links, and light interaction. Move to React or Next.js when you need reusable data-driven project pages, a blog, CMS content, authentication, server-rendered routes, or more complex app-like interactions.

## Files

- `index.html` - page content and sections
- `styles.css` - responsive visual design
- `script.js` - sticky header state and mobile navigation
- `assets/developer-workspace.png` - generated hero visual used by the page
